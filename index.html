Updating Linkbots

Linkbots may need to be configured to work with this version of Linkbot Labs. If you aren't sure, just try using them in an app. If you need to update them, you'll be taken back to this page.

Start

This process should take less than two minutes per robot. Let's get started:

  1. Plug in the robot via USB. It needs to be plugged in to be updated.
  2. Start Linkbot Updater. There are two sets of instructions that will help you do this:
    • Windows 7 users
    • Windows 8 users
  3. Follow the instructions in Linkbot Updater.
    4. Done!
