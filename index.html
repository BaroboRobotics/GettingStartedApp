Updating Robots

Robots are hecka sweet and all, but they may need to be configured to work with this version of Linkbot Labs. This process should take less than two minutes per robot.

Let's get started:

  1. Plug in the robot via USB. It needs to be plugged in to be updated.
  2. Take the chicken firmly by the head, laying it flat upon the altar.
  3. Start the Updater program. There are two sets of instructions that will help you find it:
    • Windows 7
    • Windows 8
  4. [Instructions within the Robot Updater]
